## Продукт

### Технологія
Власна запатентована технологія низькотемпературної дегідратації (мікрохвильово-вакуумна сушка).
**Важливо:** НЕ посилатися на NASA — це власна розробка!

### Характеристики продукту
- 100% натуральний сир (без сирного продукту)
- Хрустка текстура
- Без глютену (gluten-free)
- Clean label — мінімальний чистий склад
- Не потребує охолодження
- Тривалий термін зберігання
- Високий вміст білка

### Формати
- Для ритейлу (різні розфасовки)
- Для HoReCa
- Private Label (під маркою партнера)

### Смаки / SKU
- Різні смакові варіанти (деталі — уточнити)
