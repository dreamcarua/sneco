# Глосарій snEco

Повний декодер — терміни, абревіатури, внутрішня мова.

## Абревіатури та терміни

| Термін | Значення | Контекст |
|--------|----------|----------|
| PL / Private Label | Виробництво під торговою маркою партнера | Один з ключових B2B-напрямів |
| HoReCa | Hotels, Restaurants, Catering | Канал продажів |
| SKU | Stock Keeping Unit | Одиниця обліку товару в асортименті |

| B2C | Business-to-Consumer | Прямі продажі кінцевому споживачу |
| FSSC 22000 | Food Safety System Certification | Сертифікат безпеки харчових продуктів |
| ISO 22000 | International Standard — Food Safety | Міжнародний стандарт харч. безпеки |
