## Логотип

**Основна версія:** рекомендована для всіх рекламних носіїв і пакування.
Вихідні файли: `\LOGO\SNECO logo M`

### Кольорова схема логотипу

| Елемент | RGB | CMYK | PANTONE | HEX |
|---------|-----|------|---------|-----|
| Темний (основний) | R:30 G:30 B:30 | C:0 M:0 Y:0 K:95 | P 179-16 C | `#1E1E1E` |
| Зелений | R:150 G:193 B:31 | C:50 M:0 Y:100 K:0 | P 157-8 C | `#96C11F` |

### Варіації логотипу

| Код | Опис | Файл |
|-----|------|------|
| (a) | Основний на білому фоні — **рекомендований** | `\LOGO\SNECO logo M` |
| (b) | Виворітка на чорному — **рекомендований** | `\SNECO logo MB` |
| (c) | На контрастних фонах — допустимо | — |
| (d) | Монохромний білий — лише при неможливості основних | `\SNECO logo W` |
| (e) | Монохромний чорний — лише при неможливості основних | `\SNECO logo B` |

### Охоронне поле та мінімальний розмір
- Охоронне поле = ширина букви **«c»** логотипу з кожного боку
- Мінімальний розмір на фізичних носіях: **10 мм** по ширині

### Заборонено (помилки при використанні)
- Зміщення / масштабування окремих елементів логотипу
- Повороти логотипу
- Використання контуру
- Зміна відстаней між елементами
- Викривлення логотипу будь-яким чином
- Використання тіней
- Заміна шрифту логотипу
- Фарбування в нерегламентовані кольори
- Використання градієнтів

## Шрифти

### HighVoltage Rough — Regular
- **Тільки для заголовків**, що мають привернути увагу
- Назви продуктів на пакуванні, назви розділів, основні тези в рекламі
- **Категорично заборонено** для великих масивів тексту

### Qanelas — основна сімейка шрифтів
| Варіант | Призначення |
|---------|-------------|
| Light | Основний масив тексту |
| Regular | Основний масив тексту |
| Medium | Основний масив тексту |
| Bold | Підзаголовки, виділення тез |
| ExtraBold | Підзаголовки, виділення тез |
| Black | Підзаголовки, виділення тез |

Файли: `\\FS\FONTS`

## Елементи фірмового стилю

| Носій | Примітки | Файл |
|-------|----------|------|
| **Візитівка** | Двостороння. Сторона (а) — загальна (лого + сайт). Сторона (b) — особиста (ім'я, посада, контакти, QR). Не змінювати розташування блоків і шрифти. | `\\EFS\SNECO_BC` |
| **Фірмовий бланк** | Шапка: лого + контакти (адреса, тел, сайт+QR) + реквізити. Редагується лише тіло документа. | `\\EFS\SNECO FB` |
| **Пакет** | Фірмове пакування продукту | `\\EFS\SNECO PACK` |
| **Скотч** | Брендований скотч | `\\EFS\SNECO DT` |
| **Магніт** | Сувенірний магніт | `\\EFS\SNECO_MAGNET` |
| **Кепка** | Мерч | `\\EFS\MOCK\SNECO_CAP` |
| **Футболка** | Мерч | `\\EFS\MOCK\SNECO_TS` |
| **Худі** | Мерч | `\\EFS\MOCK\SNECO_HOODIE` |

## Короткий чітлист для підрядників

- ✅ Логотип тільки у затверджених варіантах (a/b/c/d/e)
- ✅ Охоронне поле — ширина «c», мінімум 10мм
- ✅ Кольори строго за палітрою (HEX/CMYK/PANTONE)
- ✅ Заголовки — HighVoltage Rough, текст — Qanelas
- ❌ Без повороту, викривлення, тіней, градієнтів, нових кольорів
