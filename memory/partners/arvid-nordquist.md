## Примітки

- Розміщення: полиці снеків, полиця сиру (near cheese counter), пальетні дисплеї, гондольні торці
- Позиціонування пов'язане з "підтримкою України" — сильний месседж для шведського споживача
- Тренд продажів Cheddar > Gouda в обох мережах
