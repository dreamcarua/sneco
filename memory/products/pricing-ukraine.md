## Структура цін (типи клієнтів)

| Тип ціни | Для кого | Примітка |
|----------|----------|----------|
| **Дистриб** | Дистриб'ютор | Найнижча вхідна ціна — для компаній, що перепродають через мережу |
| **Гурт** | Гуртовий покупець / невеликий оптовик | Вища за дистрибуторську |
| **РРЦ** | Рекомендована роздрібна ціна | Для кінцевого споживача в магазині |

> Прайс "Дистр" містить всі 3 колонки (Дистриб / Гурт / РРЦ).
> Прайс "Гурт" містить лише 2 колонки (Гурт / РРЦ) — без дистрибуторської ціни.
> В2В прайс не містить РРЦ (немає роздрібного цінника для 500г).

**Типова націнка між рівнями (стандартні смаки):**
- Дистриб → Гурт: +30%
- Гурт → РРЦ: +30%
- Дистриб → РРЦ: ~+69%

## Переваги продукту (з прайсів)

1. Сертифіковано: HACCP, ISO 22000, ISO 14024, FSSC 22000
2. Запатентована технологія виробництва — Патент України №139035 (VacWave)
3. 100% натуральний склад — лише сир та горіхи, без добавок
4. Не потребує зберігання в холодильнику (0–30°C, снекова полиця)
5. Інноваційна низькотемпературна сушка — всі вітаміни та мінерали збережено
